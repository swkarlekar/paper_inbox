# PaperInbox App Icon Assets

This zip contains:

- `AppIcon.appiconset/` — macOS app icon asset catalog set for Xcode.
- `PaperInbox_AppIcon_Master_1024.png` — the 1024×1024 master source PNG.

## How to use in Xcode

1. Open `Assets.xcassets`.
2. Replace the existing `AppIcon.appiconset` with the one in this zip, or drag `AppIcon.appiconset` into `Assets.xcassets`.
3. In the target settings, make sure the app icon source points to `AppIcon`.

The menu-bar icon should be a separate monochrome template image, not this colorful app icon.
